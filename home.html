HTML Interview Questions

1. What is HTML?
HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using elements like headings, paragraphs, links, and images. It is a markup language, not a programming language.
2. What is the difference between a Markup Language and a Programming Language?
A markup language like HTML defines the structure and presentation of data. A programming language like JavaScript performs logical operations and computations.
3. What is the use of the <a> tag in HTML?
The <a> tag is used to create hyperlinks that navigate to another webpage or resource.
Example: <a href="https://www.google.com">Visit Google</a>
4. What is the box model in CSS?
The box model defines how HTML elements are sized and spaced using:
  • Content: actual text or image
  • Padding: space around the content
  • Border: edge around the padding
  • Margin: space outside the border
5. What is the difference between id and class attributes in HTML?
  • id is unique and used to identify one element.
  • class can be shared by multiple elements for styling or behavior.
Example: <div id="header"></div> and <div class="menu"></div>
6. What are Semantic HTML Elements?
Semantic elements clearly describe their meaning in a human- and machine-readable way.
Examples: <article>, <section>, <nav>. These improve accessibility, SEO, and code readability.
